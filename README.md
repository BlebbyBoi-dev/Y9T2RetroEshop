# Y9T2RetroEshop
An E-Shop for a fictional Retro Gaming Store, for a school project

## Selected Project Scenario:
**2. Small Business E-Commerce site**
"Create an online store for a local business. Users should be able to browse products, read reviews, and make purchases."
User stories:
•	As a customer, I want to browse products by category so I can easily find what I’m looking for.
•	As a shopper, I want to read reviews so I can make confident purchasing decisions.
•	As a user, I want to securely make purchases online.

## Project Management
Project Management Tool: Obsidian.md, with the Kanban Plugin.
Sprint Plans: 2 Week sprints, over 8 weeks in total
- Sprint 1 (Weeks 1-2): Finalise idea + structure, Set up obsidian, Create UI mockup, Begin with basic HTML
- Spring 2 (Weeks 3-4): Build Homepage, Start shop page structure, add navigation header
- Sprint 3 (Weeks 5-6): Finish shop page, Build buying guide page, Start Reviews page
- Sprint 4 (Weeks 7-8): Finalise CSS, Fix bugs, Polish pages, final submission

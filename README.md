# Y9T2RetroEshop
An E-Shop for a fictional Retro Gaming Store, for a school project

## Goal:
The purpose of this project is to design and develop a user-focused e-commerce website for a retro gaming store. The website will allow users to browse games, view information, and make informed purchasing decisions 
through structured categories and reviews.

## Selected Project Scenario:
**2. Small Business E-Commerce site**
"Create an online store for a local business. Users should be able to browse products, read reviews, and make purchases."
User stories:
-	As a customer, I want to browse products by category so I can easily find what I’m looking for.
-	As a shopper, I want to read reviews so I can make confident purchasing decisions.
-	As a user, I want to securely make purchases online.

## Design Approach
The website will follow a clean, retro-inspired e-commerce design with a focus on usability, clear navigation, and consistent layout across all pages. The design will prioritise user experience by making categories and 
products easy to find.
The website is designed with a focus on accessibility, clear navigation, and consistent layout to ensure a positive user experience across all pages.

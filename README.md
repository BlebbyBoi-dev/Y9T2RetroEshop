# Y9T2RetroEshop
An E-Shop for a fictional Retro Gaming Store, for a school project

## Selected Project Scenario:
**2. Small Business E-Commerce site**
"Create an online store for a local business. Users should be able to browse products, read reviews, and make purchases."
User stories:
•	As a customer, I want to browse products by category so I can easily find what I’m looking for.
•	As a shopper, I want to read reviews so I can make confident purchasing decisions.
•	As a user, I want to securely make purchases online.
